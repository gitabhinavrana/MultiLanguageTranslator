# 🌍 Multi-language Translator with Voice Input

This app uses Facebook’s NLLB-200 model to translate text or speech into 25+ languages.

## Features
- Type text or speak using the microphone
- Translate into multiple languages
- Powered by Hugging Face Transformers + Gradio

## How to use
1. Enter text or record speech.
2. Select the target language.
3. Get instant translation.
